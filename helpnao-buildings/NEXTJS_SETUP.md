# Helpnao Buildings - Next.js Setup

## Install
```bash
npm install
```

## Run development server
```bash
npm run dev
```

Open:
```bash
http://localhost:3000
```

## Production build
```bash
npm run build
npm run start
```

## Notes
- This project was converted from Vite React to Next.js.
- Main routes kept the same:
  - `/`
  - `/buildings`
  - `/buildings/[id]`
  - `/flats/[id]`
- Existing UI, data, Tailwind styles, and client-side behavior were preserved as closely as possible.
